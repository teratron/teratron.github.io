## Перебор элементов итератора, реализация типажа или указание срока жизни

```rust
for num in 1..6 {
    println!("num = {}", num);
}
```

```rust
for i in 1..10 {
    for j in 1..10 {
        print!("{}\t", i * j);
    }
    println!();
}
```
